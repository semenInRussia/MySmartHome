# MySmartHome

_Автор_: Семён Храмцов

_Направление_: Современное программирование

_Группа_: б10 (2025)

_Дата_: 29.11.2025

Моя домашка("Мой Умный Дом") по предмету ООП в СПбГУ МКН первый семестр направления Современное Программирование.

Задача домашки была дописать код для модели умного дома

Точно были выполнены требования из [requirements.docx](requirements.docx)

Классы реализованы в файлах `EventLogger.cs`, `Program.cs`, `SmartHomeController.cs`, `AirConditioner.cs`, `Heater.cs`, `ISmartDevice.cs`, `Light.cs`

Для удобства использования классы располагаются в namespace-ах `MySmartHome` и `MySmartHome.Devices`.

```shell
dotnet run
```
